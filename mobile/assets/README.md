# Assets Directory

This directory contains images and other assets for the Subdibuddy mobile app.

## Required Assets

### App Icon (`icon.png`)
- **Size:** 1024x1024px
- **Format:** PNG
- **Usage:** Main app icon for iOS and Android

### Splash Screen (`splash.png`)
- **Size:** 1242x2436px (or similar aspect ratio)
- **Format:** PNG
- **Usage:** App launch screen

### Adaptive Icon (`adaptive-icon.png`)
- **Size:** 1024x1024px
- **Format:** PNG
- **Usage:** Android adaptive icon foreground

### Favicon (`favicon.png`)
- **Size:** 48x48px (or 32x32px)
- **Format:** PNG
- **Usage:** Web app favicon
